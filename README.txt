Latitude 26° Cape Coral Waterway & Gulf Access Guide

FILES INCLUDED
- index.html
- styles.css
- script.js
- Canals.geojson

HOW TO TEST
1. Upload all four files into the same folder on a web host.
2. Open index.html.

IMPORTANT
The map will not work correctly if you simply double-click index.html on your computer, because browsers often block local GeoJSON loading. It needs to be hosted online or opened through a local server.

BEST HOSTING OPTIONS
- Cloudflare Pages
- GitHub Pages
- Your web developer's hosting account
- A subfolder on homesbymelodie.com if Agent3000 allows file hosting

AGENT3000 EMBED CODE
Once hosted, paste this into the Agent3000 Source editor:

<iframe src="YOUR_HOSTED_MAP_URL_HERE" width="100%" height="1100" style="border:0; width:100%; max-width:100%;" loading="lazy"></iframe>

Replace YOUR_HOSTED_MAP_URL_HERE with the live URL.
